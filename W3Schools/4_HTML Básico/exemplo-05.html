Como colocar links em um código HTML

O atributo href especifica o destino do link:
Visit W3Schools

A sintaxe funciona:

<a href="URL">Descrição do link</a>

href significa (Hypertext Reference) que é o endereço de destino do link

Este é o único atributo obrigatório para âncoras definindo uma ligação para um hipertexto.
Ele indica o alvo do link, seja uma URL ou um fragmento de URL. Um fragmento de URL é
um nome precedido por uma cerquilha (#), a qual especifica um destino interno (um ID)
dentro do documento atual. URLs não precisam se limitar à documentos Web baseados
em HTTP. URLs podem usar qualquer protocolo suportado pelo navegador. Por exemplo, file,
ftp, e mailto funcionam na maioria dos navegadores.

Nota: Você pode usar o fragmento especial "top" para criar um link para o topo
da página; por exemplo Volte ao topo . Este comportamento é
especificado pelo HTML5.


Também é possivel utilizar o href para direcionar a um envio de email por exemplo, Enviar e-mail

Neste caso a sintaxe é a seguinte:

<a href="mailto:emaildodestinatario@provedor.com?Subject=Assunto do email">O que vai aparecer no navegador</a>


Também é possível enviar para mais de um endereço de e-mail:

<a href="mailto:primeiro@email.com?cc=segundo@email.com&bcc=terceiro@email.com &subject=Titulo que aparecerá no assunto do email">Texto do link que aparecerá no navegador</a>

Podemos também vincular um Javascript:
exemplo

<a href="javascript:alert('Hello World!');">O que vai aparecer no navegador</a>Executar Javascript
