Vamos ver como exibir imagens em uma página WEB

Imagens HTML são definidas com a TAG (img). O arquivo de origem (src), texto alternativo (alt), width "largura" e height "altura" são fornecidos como atributos:

<img src="nome_do_arquivo.jpg" alt="Texto alternativo" width="largura" height="altura">
