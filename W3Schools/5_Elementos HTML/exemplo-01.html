Vamos estudar sobre os elementos HTML

Um elemento HTML geralmente consiste de um início etiqueta e uma extremidade etiqueta, com o conteúdo inserido entre:

< Tagname > Conteúdo vai aqui ... < / tagname >

TAG de início Conteúdo do elemento TAG final
<h1> Meu primeiro h1 </h1>
<p> Meu primeiro parágrafo </p>
<br>    

elementos HTML sem conteúdo são chamados de elementos vazios.
Os elementos vazios não tem uma marca de fim, tal como o elemento (br) (o que indica uma quebra de linha).
